# AIAgentCogNest

AIAgentCogNest(AI + Agent + Cognition + Nest)主要记录大语言大模型（LLMs） 算法（应用）工程师相关的知识、从零开始的智能体原理及落地实践教程。欢迎有兴趣的小伙伴加入我们，共同推进AI Agent的知识体系建立和完善。
后续将会持续更新～～～


# 仅供学习，不做推广

非常感谢社区伙伴们的反馈。这个项目不知为何被传播到了技术论坛和抖音上。说实话，我自己也非常意外、蒙蔽状态，完全不清楚是被谁、通过什么方式推广出去的。

我本人从未主动进行过任何推广，也没有做推广的打算。这个项目的初衷很简单：只是想把我在学习、内部培训、实战过程中积累的笔记，以及和朋友们交流算法时总结的经验，系统地整理出来，作为自己建立知识体系的一份教程材料。

所以，它仅用于学习交流，并不想被包装或发展成一个完整的应用。希望大家能理解。
