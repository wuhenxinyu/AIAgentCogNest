# AIAgentCogNest

AIAgentCogNest(AI + Agent + Cognition + Nest)主要记录大语言大模型（LLMs） 算法（应用）工程师相关的知识、从零开始的智能体原理及落地实践教程。欢迎有兴趣的小伙伴加入我们，共同推进AI Agent的知识体系建立和完善。
后续将会持续更新～～～


# 关于这个项目
这个项目的初衷很简单：将我个人的学习笔记、内部技术分享的沉淀、团队实战AI应用中的经验，以及在工作中与朋友们、同行们交流LLM Agent应用时碰撞出的心得，系统地整理成一份教程，作为构建知识体系的基石。

# 特别说明
这里记录的，并非我一人之功。许多内容都源于同行伙伴们的共同智慧与贡献 —— 有的是工作中并肩作战的同事，有的是技术交流时慷慨分享的朋友。正是大家的参与，才让这些零散的片段，逐渐汇聚成这份可供参考的资料。

# 使用声明
这份教程仅用于学习交流。我们无意将其包装或发展成完整的商业应用，只愿它能像一本集体创作的技术手记，供有需要的朋友随时翻阅、参考。
